The build tools are written in JudoScript language.

To build Jamaica, install JudoScript first, which easy:

  1. Launch http://www.judoscript.com/download.html,
  2. download judo.jar.zip and unjar it into judo.jar.
  3. Add judo.jar to your CLASSPATH. That's it.

Then, run

  java judo -q build.judo

This script is menu driven.
